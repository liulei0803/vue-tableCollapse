# vue-tableCollapse
Put this component into table-tfoot-tr-td, then click to collapse the table-tbody contents.

### Usage:
  
    <table>
      <tbody>
        ...
      </tbody>
      <tfoot>
      <tr>
        <td>
          <collapse-table :list="dataList"></collapse-table>
        </td>
      </tr>
      </tfoot>
    </table>
